# ADR 002: 숏 포지션을 이번 포트폴리오 범위에서 제외한 이유

## 결정

이번 포트폴리오 범위에서는 숏 포지션을 허용하지 않는다.

## 이유

- 숏 회계, 차입, 마진, 담보 관리까지 포함하면 모델 복잡도가 급격히 증가한다.
- 현재 포트폴리오 목표는 현물 주식형 전략의 주문/리스크/정산 백엔드를 보여주는 것이다.
- 단순 SELL조차 보유 수량 범위 내에서만 허용하는 편이 리스크 통제 스토리를 명확하게 만든다.

## 결과

- `SELL` 주문은 available long position 범위 내에서만 승인된다.
- realized/unrealized PnL 계산도 롱 중심 단순 모델을 유지한다.
- 향후 확장 시 별도 ADR로 숏 회계 정책을 분리할 수 있다.
